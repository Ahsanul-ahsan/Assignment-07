# 🧑‍🤝‍🧑 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern React-based web application designed to help users manage and maintain meaningful friendships. It allows users to track friends, log interactions (Call/Text/Video), and visualize relationship activity using analytics.

## 📸 Project Overview

KeenKeeper helps users:
- Track friends and their contact history
- Record interactions like calls, texts, and video chats
- Visualize friendship engagement through charts
- Maintain consistency in relationships

---

## 🛠️ Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- DaisyUI
- React Icons
- Recharts
- React Toastify
- Context API
- Vite

---

## ✨ Key Features

### 🏠 1. Home Page
- Displays all friends in a responsive grid layout
- Each friend card shows:
  - Profile picture
  - Name
  - Tags
  - Days since last contact
  - Relationship status (color coded)
- Click on a friend to view detailed profile

---

### 👤 2. Friend Details Page
- Full profile information
- Relationship stats (days since contact, goal, next due date)
- Quick Check-In actions:
  - 📞 Call
  - 💬 Text
  - 🎥 Video
- Each action logs an interaction to Timeline
- Toast notification on every action

---

### 📜 3. Timeline Page
- Displays history of all interactions
- Supports filtering:
  - All
  - Call
  - Text
  - Video
- Shows interaction type, name, and date

---

### 📊 4. Stats / Analytics Page
- Pie chart visualization using Recharts
- Displays:
  - Total Calls
  - Total Texts
  - Total Videos

---

### 🧭 5. Navigation
- Fully responsive Navbar
- Active route highlighting
- Mobile-friendly dropdown menu

---

### ⚡ 6. Context API State Management
- Global timeline state handled using Context API
- All interactions shared across pages

---

### ❌ 7. 404 Page
- Custom error page for invalid routes
- Includes navigation back to Home

---
